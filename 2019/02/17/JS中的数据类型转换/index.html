JS中的数据类型转换

2019-02-17

JS

​ 在JS中，总共有7种数据类型：number、boolean、string、symbol、null、undefined、object。由于null、undefined、object、symbol等转换并没有实际上的意义，下面谈谈number、boolean、string三种类型的转换。

String

  • String()
  • 10 + “”
  • toString()

这里需要解释一下。1.toString()报错的原因，在于JS引擎无法识别这里的“ . ”到底是什么意思，即它是有歧义的。它出现在这里，有可能是表示引用toString()方法，也有可能是表示1是个浮点数。所以解决问题的关键在于避免歧义，可以用这些方法：

  • 小括号包裹数字，表示这是个整体，后面的“ . ”就是引用方法的意思。
  • 加两个点，前一个表浮点数，后一个表示引用。
  • 1.0.toString()

Number

  • Number()
  • parseInt()

parseInt()工作原理是从左往右扫描，直到首次遇见非数字的字符，然后将前面的部分转化为数字。

  • parseFloat()

parseFloat()工作原理与parseInt()基本一致，区别在于对“ . ”符号的兼容性。

  • “100” - 0
  • + “100” //100

Boolean

  • Boolean()
  • !!

两次取反后，结果与if(condition)中的结果是一致的。
