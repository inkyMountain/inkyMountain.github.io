如何实现一个 bind Pollyfill

2019-08-31

手写系列

当执行 new 的时候，到底发生了哪些事情？

  1. 构造一个空对象。
  2. 将 function 的 this 绑定到这个空对象上。
  3. 将空对象的 __proto__ 指向 function 的 prototype 上。
  4. 执行这个 function。
  5. return 上面构造的对象，除非 function 已经返回了一个对象。
