Dom元素常用尺寸

2019-05-18

Dom

以下都是element的属性

内容整理自 这个GitHub Issue

div内部滚动条相关高度

offsetHeight

元素本体高度，不包含margin, 包含 border与padding。

scrollHeight

存在滚动条时，子元素的整体高度。

scrollTop

子元素超出父元素的高度

offsetTop

与上面三个不是一家人。它是子元素距离爸爸顶部的距离。

element.getBoundingClientRect()

该方法获取的对象，属性解释如下：

  • top 距离视口 viewport 顶部的距离。
    • 元素在viewport上面时，top < 0
    • 元素在viewport下面时，top > innerHeight
    • 可以用于判断元素是否在viewport中
  • left 距离视口 viewport 左侧的距离
  • x 即x坐标，等同于 left
  • y 即y坐标，等同于 top
  • width 宽度
  • height 高度
