Flex布局常用总结

2019-03-12

Flex

父元素属性控制item的排列规则，子元素属性控制item的缩放 以及个体排列。

用于父元素的属性

==默认以flex-direction为row时的情况==

flex-direction

  • row 左 => 右 (default)

  • row - reverse 右 => 左

  • column 上 => 下
  • column - reverse 下 => 上

flex-wrap

  • nowrap 不换行 (default)
  • wrap 上 => 下换行
  • wrap-reverse 下 => 上换行

justify-content

  • flex-start
  • flex-end
  • center
  • space-between
  • space-around

align-item

  • flex-start
  • flex-end
  • center
  • base-line
  • stretch

align-content

相当于多行情况下的align-item，单行时不起作用。

用于子元素的属性

flex-grow

默认0，不放大。> 0 则在有空位时放大。

flex-shrink

默认1，即空间不够时缩小。
为0也会缩小，但是缩小的底线会高一些。

item的flex-shrink值不同时，越大缩小得越多。这一点与flex-grow类似。

flex-basis

默认auto，代表该item占据主轴的空间，用于计算是否有剩余空间。

align-self

特殊化该item对齐方式，可覆盖align-items属性

order

可用于改变item出现顺序。
